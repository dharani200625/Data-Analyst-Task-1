# Netflix Movies and TV Shows - Data Cleaning and Preprocessing

## Objective
The objective of this task is to clean and preprocess the Netflix Movies and TV Shows dataset using Python (Pandas).

## Tools Used
- Google Colab
- Python
- Pandas

## Dataset
- Netflix Movies and TV Shows Dataset (Kaggle)

## Data Cleaning Steps
- Loaded the dataset using Pandas.
- Checked for missing values.
- Handled missing values using appropriate methods.
- Checked for duplicate records (no duplicates found).
- Renamed column headers to a clean and consistent format.
- Standardized text values by removing extra spaces.
- Converted the `date_added` column to datetime format.
- Verified data types.
- Saved the cleaned dataset as `cleaned_netflix_dataset.csv`.

## Files Included
- `cleaned_netflix_dataset.csv`
- `Data_Analyst_Task_1.ipynb`
- `README.md`

## Outcome
The dataset was successfully cleaned and prepared for further data analysis.
